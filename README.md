# AEON Networking Tools

AEON Networking Tools is a Windows-friendly desktop network utility built with Electron and a local Node.js server.

It includes:

- Internet connectivity checks
- Download speed benchmarking
- Ping latency and packet loss
- DNS lookups
- TCP port probes
- HTTP reachability checks
- Traceroute diagnostics
- Local adapter and IP inspection

## Run

```powershell
npm install
npm start
```

## Validate

```powershell
npm run check
```

## Build Windows App

```powershell
npm run dist
```
